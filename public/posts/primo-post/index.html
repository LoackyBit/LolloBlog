Come Ho Creato il Mio Blog con Hugo e il Tema Stack

Ciao a tutti! In questo post condividerò come ho costruito il mio blog usando Obsidian, Hugo, GitHub e Hostinger, seguendo un tutorial di NetworkChuck e personalizzandolo con il tema Stack (https://stack.jimmycai.com/). Vi guiderò anche attraverso la configurazione del tema Stack per creare un blog elegante e funzionale.

Perché un Blog?

Ispirato da Building a Second Brain di Tiago Forte, ho iniziato questo blog per condividere le mie idee e organizzare i miei pensieri in modo accessibile e prezioso. Il mio obiettivo è esprimere quello che sto imparando e fornire spunti per me stesso, i miei amici, la famiglia e chiunque sia interessato alla mia prospettiva.

Come Ho Costruito il Sito

Ecco i passaggi principali che ho seguito per creare il mio blog, adattati dal tutorial di NetworkChuck

.

1. Scrivere Post con Obsidian

Uso Obsidian, una potente app per prendere appunti, per scrivere i miei post del blog in markdown. Tutti i post sono memorizzati in una cartella post all’interno del mio vault di Obsidian. Ogni post inizia con un blocco di front matter, che include metadati come titolo, data e tag. Per esempio:

+++
title = "Il Mio Primo Post"
date = 2025-06-13
draft = false
tags = ["test", "blog"]
+++

Obsidian mantiene le mie note private, e solo i file nella cartella post vengono pubblicati. Per assicurarmi che le immagini funzionino (anche se sto ancora risolvendo questo problema), prevedo di usare uno script Python per sincronizzare le immagini dalla cartella attachments di Obsidian alla cartella static/images di Hugo.

2. Generare il Sito con Hugo

Hugo è un generatore di siti statici che trasforma file markdown in un sito web. Ecco come l’ho configurato:

  • Installazione: Ho installato Hugo sul mio Mac usando Homebrew (brew install hugo). Ho anche installato Git e Go come prerequisiti.

  • Creazione del Sito: Ho creato un nuovo sito Hugo con hugo new site NomeBlog.

  • Configurazione del Tema: Ho scelto il tema Stack (https://stack.jimmycai.com/) per il suo design pulito in stile card. L’ho installato con:

    git submodule add https://github.com/CaiJimmy/hugo-theme-stack themes/hugo-theme-stack

  • Configurazione: Ho configurato Hugo in hugo.toml usando il comando nano per utilizzare il tema Stack:

    theme = "hugo-theme-stack"

3. Configurazione del Tema Stack

Il tema Stack offre molte opzioni di personalizzazione. Ecco le configurazioni principali che ho implementato:

Configurazione Multilingue

Ho configurato il supporto per italiano e inglese:

DefaultContentLanguage: it
languages:
    it:
        languageName: Italiano
        title: LolloBlog
        weight: 1
        params:
            sidebar:
                subtitle: Il mio blog di tecnologia e programmazione
    en:
        languageName: English
        title: LolloBlog
        weight: 2
        params:
            sidebar:
                subtitle: My technology and programming blog

Personalizzazione della Sidebar 

sidebar:
    emoji: 💻
    subtitle: Il mio blog di tecnologia e programmazione
    avatar:
        enabled: true
        local: true
        src: avatar.jpg

Widget della Homepage

Ho configurato vari widget per la homepage:

widgets:
    homepage:
        - type: search
        - type: archives
          params:
              limit: 5
        - type: categories
          params:
              limit: 10
        - type: tag-cloud
          params:
              limit: 10

4. Hosting e Deployment

  • GitHub: Uso GitHub per il controllo versione e per memorizzare il codice sorgente del blog.
  • Vercel/Hostinger: Il sito viene deployato automaticamente tramite Vercel o Hostinger ogni volta che pusho modifiche su GitHub.

Caratteristiche del Tema Stack

Il tema Stack offre molte funzionalità moderne:

  • 📱 Design Responsive: Si adatta perfettamente a dispositivi mobile e desktop
  • 🌙 Modalità Scura: Supporto nativo per tema scuro/chiaro
  • 🔍 Ricerca Locale: Ricerca veloce tra tutti i post
  • 📸 Supporto Immagini: Integrazione con PhotoSwipe per le gallerie
  • 🏷️ Gestione Tag e Categorie: Organizzazione avanzata dei contenuti
  • 📊 Indice dei Contenuti: Navigazione automatica nei post lunghi
  • ⚡ Prestazioni: Caricamento veloce senza framework pesanti

Prossimi Passi

Sto lavorando per:

  1. Automatizzare la sincronizzazione delle immagini da Obsidian
  2. Migliorare il SEO con meta tag ottimizzati
  3. Aggiungere un sistema di commenti per l’interazione con i lettori
  4. Creare più contenuti su programmazione e tecnologia

Conclusione

Costruire questo blog è stato un processo gratificante che mi ha permesso di combinare strumenti potenti per creare una piattaforma personale per condividere conoscenze. Il tema Stack di Hugo fornisce un’eccellente base per un blog moderno e professionale.

Se hai domande o vuoi suggerimenti per migliorare il blog, sentiti libero di contattarmi sui social media linkati in fondo alla pagina!
